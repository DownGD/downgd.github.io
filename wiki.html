downgd.github.io

Download a single folder from a GitHub repository

GitHub repositories often contain hundreds or thousands of files. However, GitHub does not provide a native way to download just one directory from a repository.

downgd.github.io is a lightweight web tool designed to solve this limitation by allowing users to download only the directory they need.

The problem

When working with large repositories, cloning or downloading the entire project can be slow and unnecessary. This is especially frustrating for students and developers who only need a few configuration files or assets.

The solution

This tool fetches all files inside a specified GitHub directory and packages them into a ZIP file automatically. No authentication, plugins, or setup required.

Usage

To start a download, pass the GitHub directory URL as a url parameter.

downgd.github.io?url=https://github.com/mrdoob/three.js/tree/dev/build

The download will begin immediately after opening the link.

Custom filenames

You can optionally specify the output ZIP filename by adding the filename parameter.

downgd.github.io?url=https://github.com/mrdoob/three.js/tree/dev/build&filename=three-js-build

Advantages

  • Download only what you need
  • No repository cloning
  • Minimal interface, no distractions
  • Works directly in the browser

Alternatives

Tools like GitZip and DownGit offer similar functionality, but often include unnecessary UI elements or require extensions. This project focuses on simplicity.
